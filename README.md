# Диплом HSE 

Тема: ИИ-ассистент на основе специфической базы знаний  
Автор: Распитин Илья Андреевич  
  
План работ:  
- Формализация функциональных требований  
- Формирование базы знаний и структурирование ее для ИИ-ассистента. Данный пункт включает в себя:  
  - Поиск источника/-ов для базы знаний  
  - Нормализация данных (при необходимости)  
  - Структурирование/разделение данных  
- Подготовка языковой модели ии ее дообучение по технологии LangChain
- Реализация остальных компонентов системы, включая алгоритмы обработки текста и интеграции с мессенджером Telegram
- Проведение тестирования системы на предмет корректности работы
- Документирование системы
- Подготовка пояснительной записки и презентации дипломной работы
